# concurrent-programming-for-java

## 初衷(Original Intention)

- 整理下自己所学
- 带动更多的人进入 Java 并发的世界
- GitHub 项目地址，欢迎 `Fork`、`Star`：<https://github.com/junicorn/concurrent-programming-for-java>
- GitBook 讲解，采用 Maven 构建，分模块学习，涉及多线程基础、多线程设计模式、JDK并发包详细介绍、并发编程深入等知识。
- **拥有初学者的心态是件了不起的事情**
